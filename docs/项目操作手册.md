# BananaPod 使用说明（给第一次使用的人）

BananaPod 是一个用来生成和编辑图片的工具。  
你可以把它当成一块「智能画板」，写一句文字说明，就能让它帮你出图、改图、做对比图。

---

## 一、我能用 BananaPod 做什么？

你可以把 BananaPod 理解为一块「智能画板」，它可以帮你：

- 根据文字描述生成图片  
  例子：输入「黄昏的城市街景，霓虹灯反射在雨后的地面」，点击生成，就会出现对应风格的图片。
- 调整现有图片  
  例子：把一张室内图改成「夜景氛围」，或者让背景更干净、更有设计感。
- 做简单排版  
  - 在画布上放多张图片
  - 加文字说明、标题
  - 调整位置、大小、透明度等
- 做「天气 / 场景」快速尝试  
  通过底部的香蕉按钮，快速切换「晴天、黄昏、夜景」等不同氛围，找感觉。

如果你做的是方案图、海报、汇报插图或灵感草图，BananaPod 都可以帮到你。

## 二、如何打开 BananaPod？

项目可以有不同的打开方式，这里讲两种你最可能遇到的情况：

### 2.1 如果你已经拿到一个网址

1. 打开电脑浏览器（推荐 Chrome / Edge）。
2. 在地址栏输入给你的链接，例如：  
   `http://localhost:3001/` 或者某个公司内部地址。
3. 回车后，看到中间是画布、下方有输入框和一个香蕉图标，就说明打开成功。

### 2.2 如果你拿到的是一个应用程序（例如 exe 文件）

有时候开发同学会打包成一个桌面应用给你：

1. 找到他们发给你的安装包或绿色版程序（例如名字里带有 BananaPod）。
2. 双击运行，按提示完成安装（或直接打开）。
3. 打开后界面和网页版本类似，中间是画布，底部有输入框。

> 如果你什么都没有拿到（没有网址、也没有程序），可以让开发同学参照 README 里的说明帮你搭好环境，这一部分不需要你自己操作。

## 三、界面简单认识一下

不需要记住所有细节，你只要知道这几个区域就够用：

- 中间：**画布区域**  
  放图片、文字和各种图形，你看到的主要内容都在这里。

- 左上或左侧：**工具区域**  
  选择画矩形、圆形、箭头、文字等。

- 右侧：**属性区域**  
  选中一个元素后，可以在这里改颜色、大小、透明度等等。

- 最底部：**提示词输入栏（PromptBar）**  
  像聊天框一样的输入框，用来写「让 AI 生成或修改图片」的文字说明。

- 提示词输入栏右侧：**香蕉按钮**  
  点它会打开一个「提示词小实验室」，可以在里面挑选或修改提示词。

首次使用时，可以配合以下三张截图理解界面布局：

## 四、最常用的三个操作

### 4.1 从文字生成一张图片

1. 在底部输入框中输入一句清楚的描述，例如：  
   「现代风客厅，落地窗，黄昏阳光，暖色调」
2. 选择合适的模式（一般默认就可以，找不到就请开发同学帮看一次）。
3. 点击旁边的生成按钮。
4. 等待几秒，画布中会出现一张新图片。

小建议：

- 尽量写「谁 + 在哪里 + 做什么 + 氛围如何」，效果更稳定。
- 不要一句话塞太多要求，可以分几次尝试。

### 4.2 修改一张已有图片

1. 把图片拖进画布，或者用界面里的「上传图片」按钮添加。
2. 点击这张图片，让它处于选中状态（一般会出现边框或选中效果）。
3. 在底部输入框写上你想修改的方向，例如：  
   「把氛围改成雨天夜景，保留原来的构图」  
4. 点击执行/生成按钮。
5. 生成后可以选择保留原图、新图，或让新图覆盖原图，具体由界面提示决定。

### 4.3 做一组对比图或说明图

1. 在画布上放多张图片（多次生成，或拖进多张图片）。
2. 使用对齐和缩放操作，让图片排成整齐的栅格或拼贴。
3. 用文字工具在旁边加简单说明，例如「方案 A」「方案 B」「优化前 / 优化后」。
4. 最后导出整张画布作为一张完整的对比图。

### 4.4 使用「快捷效果」（不会写提示词时）

如果你一时不知道该怎么写提示词，可以试试底部输入框右侧的「快捷效果」按钮：

1. 在画布下方找到输入框，右侧会有一个小图标按钮（鼠标停上去会显示「选择一个快捷效果」类似的提示）。
2. 点击这个按钮，会弹出一个小面板：
   - 上半部分是「我的效果」：如果维护同学帮你预先存了一些常用语句，点名字就会自动把对应文字填到底部输入框。
   - 下半部分是系统内置的「快捷效果」列表，比如「擦掉某个物体」「在某个位置加东西」「提升清晰度」等。
3. 点击任意一个效果，对应的文字会自动出现在底部输入框中。
4. 如果文字里有方括号（例如「[物体描述]」「[具体位置]」），可以简单改成你自己的内容，再点击生成即可。

## 五、使用香蕉按钮 Prompt 实验室

香蕉按钮是帮助你「想提示词」的助手，适合这样的人：

- 不知道怎么写提示词的人。
- 想快速切换不同天气 / 氛围效果的人。

### 5.1 打开和整体布局

1. 在页面底部输入框右侧，找到一个香蕉图标按钮。
2. 点击后，会在输入框上方弹出一个面板：
   - 上半部分：一个网页区域，用来浏览 LeaderAI 提示词示例。
   - 下半部分：一排「天气 / 场景」卡片，例如「晴天、黄昏、夜景」。

这个面板会自动避开底部输入框，不会挡住你正在写字的地方。

### 5.2 使用「天气 / 场景」卡片

1. 在弹窗下半部分找到想要的氛围，例如「晴天」「黄昏」「雪景」。
2. 点击卡片，对应的提示词会自动填到底部输入框里。
3. 弹窗不会因此关闭，你可以继续点击其他卡片反复尝试。
4. 如果觉得下面卡片占地方，可以点击标题旁的小箭头，把卡片先收起来；需要时再点一次展开。

### 5.3 从网页里「抄」提示词

1. 在弹窗上半部分的网页中，用鼠标像平时一样选中文字。
2. 按快捷键复制（Ctrl + C）。
3. 只要鼠标还停留在这个弹窗范围内，系统会自动把你刚复制的文字填到底部输入框里。
4. 如果你离开了弹窗，系统就不再盯着你的剪贴板，不会打扰你在别的地方复制内容。

如果自动提取没有生效，弹窗顶部还有一个「粘贴」小按钮，可以手动把当前剪贴板里的文字填到输入框。

### 5.4 它做了哪些「安全保护」？

简单理解为：

- 内嵌网页只能在这个弹窗里活动，不能随便跳出去干别的。
- 像「复制并尝试」这类可能会触发外部操作的按钮，会被限制，防止误点造成未知后果。

你可以放心当成「提示词词典」来用。

## 六、如何选择图像模型？

在大多数情况下，你可以直接使用默认模型，不用特意修改。  
如果你想更精细地控制效果，可以按照下面的思路来选择：

- **在哪里修改？**  
  打开界面右上方或顶部工具栏里的「设置」按钮，在弹出的设置面板中找到「图像模型」区域，会看到几张模型卡片，每张卡片上写着模型名称和大致价格。

- **推荐的选择方式**  
  - **日常草图 / 灵感尝试**：  
    适合选择价格较低、响应较快的模型，例如：`gemini-2.5-flash-image`（约 ¥0.08/次）。  
    用来快速试想法、做多轮尝试比较划算。
  - **重要方案 / 需要更精细的画面**：  
    可以选择质量更高但稍贵的模型，例如：`gemini-3-pro-image-preview`（约 ¥0.3/次）。  
    适合最终方案图、需要更多细节时使用。
  - **团队内部模型（nano-banana 系列）**：  
    如果界面里有 `nano-banana` 或 `nano-banana-2` 等模型，一般代表是团队接入的内部或风格更稳定的模型，价格和计费方式可以向维护同学确认，适合批量生成、风格统一的需求。

- **不确定怎么选时**  
  - 先用默认模型完成 80% 的工作；  
  - 真正需要「精修」的几张图，再切换到更高质量的模型生成一遍即可。

## 七、如何导出作品

BananaPod 的导出方式可能会根据版本稍有不同，大致有几种常见方式：

- 导出整张画布为一张图片（常见格式如 PNG）；
- 导出部分区域（选中某些元素后导出）；
- 右键或菜单中提供「导出 / 下载」选项。

如果你在界面中找不到导出按钮，可以：

1. 先问一下开发或维护同学，看当前版本是否已经接入导出功能。
2. 或者用系统截图工具（不推荐长期使用，但紧急情况下可应急）。

## 八、遇到问题时怎么描述更清楚？

当你遇到问题，需要找技术同学帮忙时，可以尽量提供这些信息：

1. **你想做什么？**  
   例如：「我想把这张图改成夜景，但生成的图总是太暗。」
2. **你是在哪个位置操作的？**  
   例如：「我是在底部输入框写的提示词，点击右边的生成按钮。」
3. **最好配一张截图**  
   - 可以截取整屏，包含画布和底部输入框。
   - 如果涉及提示词，也可以把文字一并发给对方。

说清楚这三点，基本可以让对方很快复现问题。
